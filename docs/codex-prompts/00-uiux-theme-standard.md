# UI UX Pro Max Theme Standard

Use this file as the global design standard for Hooshyaran.ir.

Hooshyaran.ir must feel like a premium Persian RTL enterprise AI product catalog, not a default template.

## Core Direction

- Persian and RTL first
- Enterprise SaaS style
- Clean and premium layout
- Strong visual hierarchy
- Consistent spacing, cards, buttons, typography and sections
- Suitable for B2B product marketing
- SEO-friendly and mobile responsive

## Theme Mood

- Secure
- Controlled
- Modern
- Trustworthy
- Product-focused
- Technical but understandable

## Suggested Colors

- Primary: #213448
- Secondary: #547792
- Accent: #00B8A9
- Orange accent: #F28A00
- Green accent: #22C000
- Background: #F7FAFC
- Text: #1F2937
- Muted: #6B7280

## Logo

Use the uploaded orange and green logo or the SVG concept asset in the header, footer and brand sections. Keep it clean and balanced.

## Homepage Animation

The homepage should include tasteful lightweight animation:

- Subtle animated AI/network/grid background in hero
- Smooth entrance animation for hero text and product cards
- Animated flow concept: organization data to AI layer to workflow/dashboard
- Refined hover effects for buttons and cards
- Decorative node or connection-line animation

Animation rules:

- Prefer CSS animation
- Use JavaScript only when needed
- Respect prefers-reduced-motion
- Keep it subtle and professional
- Do not damage SEO, accessibility or performance

## Product Pages

Each product page should include:

- Product hero
- Problem section
- Solution section
- Benefits
- Public features
- Use cases
- Target audience
- FAQ
- Final CTA

Do not expose internal architecture, stack, algorithms or private product logic.

## Checklist

Before finishing any UI step, check:

- Premium enterprise look
- Full RTL support
- Mobile friendly
- Clear CTA
- SEO-safe headings and text
- Lightweight animations
- Consistent design system
